## How to Run the Project

1. Clone this repository.
2. Open MySQL Workbench (or any MySQL-compatible client).
3. Run the `database/full_database.sql` script to create the database and populate it with sample data.
4. Open the SQL scripts in the `sql/` directory and execute them individually to reproduce the analysis performed throughout the project.
5. Refer to the `screenshots/` directory for sample outputs and the `bug_reports/` and `knowledge_base/` folders for supporting documentation.
