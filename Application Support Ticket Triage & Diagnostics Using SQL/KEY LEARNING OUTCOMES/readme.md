## Key Learning Outcomes

Through this project, I strengthened my understanding of the day-to-day responsibilities of an Application/Technical Support Engineer by applying SQL to realistic support scenarios. Specifically, I gained practical experience in:

- Designing a relational database schema for a support ticketing system.
- Writing SQL queries to triage and analyse support incidents.
- Identifying recurring issues and product health trends.
- Measuring engineer workload and ticket resolution performance.
- Simulating SLA reporting using SQL.
- Converting support tickets into structured bug reports.
- Creating a knowledge base article for recurring incidents.
- Organising a technical project using Git and GitHub.
