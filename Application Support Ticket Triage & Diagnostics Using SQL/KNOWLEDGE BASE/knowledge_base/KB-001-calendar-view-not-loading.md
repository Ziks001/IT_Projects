# KB-001 – Leave Management System Calendar Does Not Load in Internet Explorer

## Issue

Users report that the calendar view does not load when accessing the Leave Management System using Internet Explorer.

---

## Symptoms

- Blank calendar page.
- Calendar controls do not appear.
- Users cannot view leave schedules.

---

## Cause

The calendar component relies on modern web technologies that are not fully supported by Internet Explorer.

---

## Resolution

Use one of the supported browsers:

- Microsoft Edge
- Google Chrome
- Mozilla Firefox

After switching browsers, refresh the application and reload the calendar.

---

## Workaround

If Internet Explorer must be used for legacy applications, open the Leave Management System using Microsoft Edge in Internet Explorer Compatibility Mode where available.

---

## Applies To

- Leave Management System
